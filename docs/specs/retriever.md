# Spec: Retriever

## Назначение

Retriever отвечает за поиск подтверждающего контекста для найденных риск-кандидатов. Его задача - дать системе не просто ответ модели, а опору на внешние и внутренние источники.

В системе есть два контура retrieval:

- `normative retrieval` - поиск по доверенным внешним правовым источникам;
- `project RAG` - поиск по локальной базе документов проекта.

## Источники

В retrieval могут попадать:

- нормативные веб-источники и реестры из allowlist;
- локальные шаблоны, политики, договоры и связанные документы проекта.

Для каждого источника должны быть доступны:

- стабильный `source_id`;
- тип источника;
- ссылка или путь;
- метаданные версии для локальных документов.

## Индекс

Локальный индекс строится по versioned chunks. Это нужно, чтобы любой найденный фрагмент можно было точно связать с исходным документом и его версией.

Единица индексации:

- `doc_id`
- `version`
- `chunk_id`
- `text`
- `metadata`

Параметры chunking:

- chunk size: `600-900` токенов;
- overlap: `80-120` токенов.

## Поиск и reranking

Поиск строится в несколько шагов:

1. формируется запрос из `risk_type`, `query`, `jurisdiction`, `time_ref` и контекста абзаца;
2. выполняется первичный hybrid search (`keyword` + `dense`);
3. найденные результаты проходят reranking;
4. в downstream-контекст попадают только самые релевантные фрагменты.

Ограничения выдачи:

- top-k после первичного поиска: `5..8`;
- в LLM-контекст передается top-3.

## Контракт

На вход retriever получает:

- `session_id`
- `query`
- `risk_type`
- `jurisdiction`
- `time_ref`
- `paragraph_id`

На выходе возвращается набор evidence-объектов со следующими полями:

- `source_id`
- `source_type`
- `snippet`
- `uri`
- `retrieval_score`
- `retrieved_at`
- `doc_id` / `version` / `chunk_id` для локального контура

## Ограничения

- timeout внешнего retrieval-вызова: `8s`;
- max external calls per request: `5`;
- пустая выдача возвращается как `retrieval_empty=true`, это не ошибка;
- недоверенные домены не попадают в выдачу даже при высокой релевантности.
